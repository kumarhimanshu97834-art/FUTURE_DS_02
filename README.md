# FUTURE_DS_02
Social Media Campaign Performance Dashboard

## Overview
This project analyzes Facebook and Instagram ad campaign data
to evaluate performance using key marketing metrics.

## Metrics Covered
- Total Impressions
- Total Clicks
- Total Conversions
- Cost per Conversion
- CTR (Click Through Rate)
- Total Spend

## Tools Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Plotly

## Output
- KPI Cards
- Campaign Performance Charts
- Single-page Dashboard Image
